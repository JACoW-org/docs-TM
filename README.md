# TM Documentation

This documentation can be consulted at [TM-docs.JACoW.org](https://TM-docs.JACoW.org). It is generated from
Markdown files using [MkDocs](http://www.mkdocs.org/) and
[Material for MkDocs](http://squidfunk.github.io/mkdocs-material/).

## Basic Usage

```sh
$ pip install -r requirements.txt

# if you want to run it locally
$ mkdocs serve

# in order to generate HTML files
$ mkdocs build
```
